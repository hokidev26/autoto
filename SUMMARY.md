## 🚀 快速開始流程

### 3 分鐘版本
```bash
bash install.sh
# 或在設定頁面 / ~/.autoto/config.json 填入 API Key
autoto
```

### 15 分鐘版本（含 LINE）
1. 執行 install.sh（5 分鐘）
2. 申請 LINE 官方帳號（5 分鐘）
3. 設定 webhook 和啟動（5 分鐘）
4. 測試成功！

- Python 3.11+
