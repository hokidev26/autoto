連接聊天平台後，你可以透過這些平台與 AutoTo 對話。開啟開關、填入 Token、儲存即可。

Discord

1. 前往 Discord Developer Portal 建立 Application
2. 左側選 Bot → 點 Reset Token 取得 Token
3. 開啟 Message Content Intent
4. 用 OAuth2 URL 邀請 Bot 到你的伺服器

LINE

1. 前往 LINE Developers 建立 Messaging API Channel
2. 在 Channel 頁面取得 Channel Secret
3. 點 Issue 取得 Channel Access Token
4. Webhook URL 設為 https://你的網域:5670/callback

Telegram

1. 在 Telegram 搜尋 @BotFather 並傳送 /newbot
2. 依照指示設定名稱，取得 Bot Token
3. 貼上 Token 並開啟開關即可，無需設定 Webhook

微信

1. 前往微信公眾平台註冊服務號或訂閱號
2. 在開發 → 基本配置取得 AppID 和 AppSecret
3. 伺服器配置 URL 設為 https://你的網域:5671/wechat

WhatsApp

1. 前往 Meta Developer Portal 建立應用
2. 新增 WhatsApp 產品，取得 Phone Number ID 和 Access Token
3. Verify Token 自訂一組密碼，Webhook URL 設為 https://你的網域:5680/webhook

Slack

1. 前往 Slack API 建立 App
2. OAuth & Permissions 加入 chat:write、app_mentions:read 權限
3. Install to Workspace 取得 Bot Token (xoxb-...)
4. Basic Information 頁面取得 Signing Secret

Messenger

1. 前往 Meta Developer Portal 建立應用
2. 新增 Messenger 產品，連結你的 Facebook 粉絲專頁
3. 產生 Page Access Token
4. Webhook URL 設為 https://你的網域:5682/webhook

QQ

1. 安裝 Lagrange 或 NapCat 等 OneBot v11 框架
2. 登入 QQ 帳號，設定 HTTP API 地址（預設 5700）
3. 設定 HTTP POST 上報到 http://127.0.0.1:5683/onebot

Instagram

1. 前往 Meta Developer Portal 建立應用
2. 新增 Instagram Graph API 產品
3. 連結你的 Facebook 粉絲專頁和 Instagram 商業帳號
4. 產生 Access Token（需 instagram_basic, instagram_manage_comments 權限）

社群發文平台

以下平台用於主動發佈貼文（照片/影片/文字），需要對應的 API Token。

Facebook

1. 前往 Meta Developer Portal 建立應用
2. 取得 Page Access Token（需 pages_manage_posts, pages_read_engagement 權限）
3. 也可複用 Instagram 的 Token（同一個 Meta App）

X / Twitter

1. 前往 X Developer Portal 建立 App
2. 在 Keys and tokens 頁面取得以上 4 組金鑰
3. App 需要 Read and Write 權限

Threads

1. 前往 Meta Developer Portal
2. 新增 Threads API 產品
3. 產生 Access Token（需 threads_basic, threads_content_publish 權限）
